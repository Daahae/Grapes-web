# Grapes
__서버프로그래밍설계 프로젝트__

## 목표
* SNS와 유사한 웹어플리케이션 개발
* Front-end: 드래그 & 드롭, 라이트박스, 무한스크롤, 반응형 웹
* Back-end: 로그인-아웃 세션유지, 본인정보 및 게시글 친구관계 DB관리
